Задание 4. DOM

Описание задания

Создать интерактивный элемент с использованием классов и событий.

При каждом четном нажатии выводится фраза класса class1, при нечетном class2.

Фразы выводятся в случайном, не повторяющемся порядке.
Когда фразы заканчиваются, то выводится alert c фразой "Фразы закончились".

По нажатию на "Перекрасить" у всех четных строк изменяется начертание на полужирное.

На странице в части aside создать нумерованный список. По нажатию на кнопку "Создать" в этом тэге фраза выводится в виде элемента многоуровневого списка.
